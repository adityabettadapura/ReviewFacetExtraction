Readme.txt
----------

1. Extract the POS tagger files.
2. Extract the input json file into the same folder where tagger files are contained.
3. Run extractfacet.py. Output stored in output.json

Input data - Amazon product reviews obtained from here:
http://snap.stanford.edu/data/amazon/productGraph/categoryFiles/reviews_Musical_Instruments_5.json

Tools used - Stanford University Parts-Of-Speech Tagger obtained from here:
https://nlp.stanford.edu/software/tagger.html
